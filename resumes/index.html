How to Print or Save as PDF

  1. Open the resume you want using the View Resume button above
  2. Click the Print / Save PDF button in the top-right corner of the resume
  3. In the print dialog, select Save as PDF as the destination (Chrome/Edge recommended)
  4. Set paper size to Letter, margins to Default or None
  5. Enable Background graphics for best results, then save
